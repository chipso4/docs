## `GET /v1/units/{serial_number}/parts`

Return the bill of materials for a unit — serviceable replacement parts with images, specifications, cross-references, and warranty coverage status.

### Path parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `serial_number` | string | **Yes** | The unit's serial number |

### Query parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `include_images` | boolean | No | Include inline image URLs (default: `true`) |
| `in_stock_only` | boolean | No | Filter to in-stock parts only |
| `serviceable_only` | boolean | No | Filter to serviceable/replaceable parts (default: `true`) |

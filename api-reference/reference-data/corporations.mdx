## `GET /v1/corporations`

Retrieve the list of authorized distributors (corporations) in your network.

### Response `200 OK`

```json
{
  "corporations": [
    {
      "corporationId": "CORP-001",
      "corporationName": "Premier Plumbing Supply",
      "extendedAttributes": []
    },
    {
      "corporationId": "CORP-002",
      "corporationName": "National Wholesale Supply Co.",
      "extendedAttributes": []
    }
  ]
}
```

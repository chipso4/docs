## `GET /v1/invoices`

Retrieve invoice headers within a date range. Returns [`CoreInvoice`](/data-types/core-objects#coreinvoice) entities (header-level summary — without line items). Supports pagination. Also used for background bulk caching of invoice data for warranty claim processing.

### Query parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `FromDate` | string (date) | No | Start of date range (ISO 8601) |
| `ToDate` | string (date) | No | End of date range (ISO 8601) |
| `page` | integer | No | Page number (default: 1) |
| `limit` | integer | No | Results per page (default: 50) |

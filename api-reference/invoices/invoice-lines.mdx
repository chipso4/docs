## `GET /v1/invoices/lines`

Retrieve line items for a specific invoice. Returns [`CoreInvoiceLineItem`](/data-types/core-objects#coreinvoicelineitem) entities with shipment tracking, vendor details, and serial numbers.

<Info>
**Prefer the details endpoint.** [`GET /v1/invoices/details`](/api-reference/invoices/invoice-details) returns the full [`CoreInvoice`](/data-types/core-objects#coreinvoice) with line items embedded. Use this endpoint only when you need line items without the invoice header.
</Info>

### Query parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `InvoiceNumber` | string | **Yes** | The invoice number |

Each item in the `items` array follows the [`CoreInvoiceLineItem`](/data-types/core-objects#coreinvoicelineitem) schema. Key nested types:

| Field | Type | Description |
|-------|------|-------------|
| `vendor` | [`CoreOrganization`](/data-types/core-objects#coreorganization) | Vendor/supplier for this line |
| `shipTo` | [`CoreAccountLocation`](/data-types/core-objects#coreaccountlocation) | Line-level ship-to (overrides header) |
| `shipments` | [`CoreLineShipment[]`](/data-types/core-objects#corelineshipment) | Shipment tracking per line |

See the [core objects reference](/data-types/core-objects) for full field listings.

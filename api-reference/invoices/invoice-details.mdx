## `GET /v1/invoices/details`

Retrieve full details for a specific invoice including line items. Returns a [`CoreInvoice`](/data-types/core-objects#coreinvoice) entity with embedded [`CoreInvoiceLineItem`](/data-types/core-objects#coreinvoicelineitem) array.

### Query parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `InvoiceNumber` | string | **Yes** | The invoice number to retrieve |

### Response schema

The response follows the [`CoreInvoice`](/data-types/core-objects#coreinvoice) schema. Key nested types:

| Field | Type | Description |
|-------|------|-------------|
| `customer` | [`CoreOrganization`](/data-types/core-objects#coreorganization) | Customer entity |
| `billTo` | [`CoreAccountLocation`](/data-types/core-objects#coreaccountlocation) | Bill-to with nested location and address |
| `shipTo` | [`CoreAccountLocation`](/data-types/core-objects#coreaccountlocation) | Ship-to with nested location and address |
| `soldTo` | [`CoreAccountLocation`](/data-types/core-objects#coreaccountlocation) | Sold-to (if applicable) |
| `contact` | [`CoreContact`](/data-types/core-objects#corecontact) | Invoice contact |
| `items` | [`CoreInvoiceLineItem[]`](/data-types/core-objects#coreinvoicelineitem) | Invoice line items |
| `items[].vendor` | [`CoreOrganization`](/data-types/core-objects#coreorganization) | Line-level vendor |
| `items[].shipTo` | [`CoreAccountLocation`](/data-types/core-objects#coreaccountlocation) | Line-level ship-to override |
| `items[].shipments` | [`CoreLineShipment[]`](/data-types/core-objects#corelineshipment) | Shipment tracking per line |

See the [core objects reference](/data-types/core-objects) for full field listings of each type.

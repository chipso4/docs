## `GET /v1/media`

Retrieve media (documents, images, manuals, diagrams) associated with a model or part. This is a unified endpoint — all media types are served from here. Query by model number for product documentation, or by part number for part-specific media.

### Query parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `model_number` | string | **Yes*** | Product model number. At least one of `model_number` or `part_number` is required. |
| `part_number` | string | **Yes*** | Part number. At least one of `model_number` or `part_number` is required. |
| `mediaType` | string | No | Filter by type: `"document"`, `"image"`, `"video"`, `"diagram"` |
| `category` | string | No | Filter by category (e.g., `"installation_guide"`, `"part_photo"`) |

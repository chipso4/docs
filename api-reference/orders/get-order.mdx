## `GET /v1/sales-order`

Retrieve full details for a specific sales order. Returns a [`CoreOrder`](/data-types/core-objects#coreorder) entity with line items and account locations.

### Path parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `orderNumber` | string | **Yes** | The sales order number |

The response follows the [`CoreOrder`](/data-types/core-objects#coreorder) schema. Key nested types:

| Field | Type | Description |
|-------|------|-------------|
| `billTo` | [`CoreAccountLocation`](/data-types/core-objects#coreaccountlocation) | Bill-to account and address |
| `shipTo` | [`CoreAccountLocation`](/data-types/core-objects#coreaccountlocation) | Ship-to account and address |
| `items` | [`CoreOrderItem[]`](/data-types/core-objects#coreorderitem) | Order line items with allocation and shipment tracking |

See the [core objects reference](/data-types/core-objects) for full field listings.

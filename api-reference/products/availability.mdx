## `GET /v1/products/availability`

Check stock availability for a product, optionally filtered to a specific warehouse location.

### Query parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `ItemId` | string | **Yes** | The product/part identifier |
| `LocationId` | string | No | Filter to a specific warehouse. If omitted, returns availability across all locations. |

### Response `200 OK`

```json
{
  "itemId": "PROD-12345",
  "locations": [
    {
      "locationId": "WH-10",
      "locationName": "Dallas Main Warehouse",
      "quantityOnHand": 24,
      "quantityAvailable": 18,
      "quantityOnOrder": 50,
      "unitOfMeasure": "EA"
    },
    {
      "locationId": "WH-20",
      "locationName": "Houston Branch",
      "quantityOnHand": 6,
      "quantityAvailable": 6,
      "quantityOnOrder": 0,
      "unitOfMeasure": "EA"
    }
  ]
}
```

| Field | Type | Description |
|-------|------|-------------|
| `itemId` | string | Product identifier |
| `locations` | array | Availability by location |
| `locations[].locationId` | string | Warehouse identifier |
| `locations[].locationName` | string | Warehouse display name |
| `locations[].quantityOnHand` | decimal | Total physical quantity |
| `locations[].quantityAvailable` | decimal | Quantity available for sale (on hand minus allocated) |
| `locations[].quantityOnOrder` | decimal | Quantity on incoming purchase orders |
| `locations[].unitOfMeasure` | string | Unit of measure |

## `GET /v1/products/pricing/customer`

Retrieve customer-specific pricing for a product. The price may differ from list price based on the customer's contract, volume tier, or promotional pricing.

### Query parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `ItemId` | string | **Yes** | The product/part identifier |
| `CustomerId` | string | **Yes** | The customer identifier |

### Response `200 OK`

```json
{
  "itemId": "PROD-12345",
  "customerId": "C-827451",
  "unitPrice": 249.99,
  "currency": "USD",
  "priceList": "CONTRACT-A",
  "unitOfMeasure": "EA",
  "extendedAttributes": []
}
```

| Field | Type | Description |
|-------|------|-------------|
| `itemId` | string | Product identifier |
| `customerId` | string | Customer identifier |
| `unitPrice` | decimal | Customer-specific unit price |
| `currency` | string | Currency code |
| `priceList` | string | Price list or contract that determined the price |
| `unitOfMeasure` | string | Unit of measure |
| `extendedAttributes` | array | Additional pricing attributes |

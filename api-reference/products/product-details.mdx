## `GET /v1/products/details`

Retrieve detailed information for a specific product by item ID. Returns a [`CorePart`](/data-types/core-objects#corepart) entity.

### Query parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `ItemId` | string | **Yes** | The product/part identifier |

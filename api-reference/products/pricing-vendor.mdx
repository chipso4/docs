## `GET /v1/products/pricing/vendor`

Retrieve vendor (cost) pricing for a product. Used to determine the distributor's cost basis.

### Query parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `ItemId` | string | **Yes** | The product/part identifier |
| `VendorId` | string | **Yes** | The vendor identifier |
| `LocationId` | string | No | Filter by warehouse location |

### Response `200 OK`

```json
{
  "itemId": "PROD-12345",
  "vendorId": "V-5010",
  "unitCost": 175.00,
  "currency": "USD",
  "unitOfMeasure": "EA",
  "extendedAttributes": []
}
```

| Field | Type | Description |
|-------|------|-------------|
| `itemId` | string | Product identifier |
| `vendorId` | string | Vendor identifier |
| `unitCost` | decimal | Vendor unit cost |
| `currency` | string | Currency code |
| `unitOfMeasure` | string | Unit of measure |
| `extendedAttributes` | array | Additional pricing attributes |

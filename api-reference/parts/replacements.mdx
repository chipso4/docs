## `GET /v1/products/replacements`

Retrieve available replacement parts for a failed part. Returns the original part (if still available), any superseding parts, and optionally non-OEM alternatives. Results include a recommendation score when available.

### Query parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `PartNumber` | string | **Yes** | The failed part number |
| `IncludeNonOEM` | boolean | No | Include non-OEM alternatives (default: `false`) |

### Replacement types

| Type | Description |
|------|-------------|
| `exact` | Same part number — direct match |
| `superseded` | A newer part that replaces the failed part |
| `compatible` | Functionally compatible part from the same manufacturer |
| `aftermarket` | Third-party replacement (only if `IncludeNonOEM=true`) |
| `universal` | Universal-fit replacement |

See [`ReplacementPart`](/data-types/parts) for the full type definition.

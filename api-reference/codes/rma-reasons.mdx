## `GET /v1/codes/rma-reasons`

Retrieve all available RMA reason codes. These classify why a product is being returned and are presented to the user during RMA creation.

### Response `200 OK`

```json
{
  "reasonCodes": [
    {
      "code": "DEFECTIVE_PART",
      "description": "Part was defective upon arrival"
    },
    {
      "code": "WRONG_ITEM",
      "description": "Incorrect item was shipped"
    },
    {
      "code": "DAMAGED_SHIPPING",
      "description": "Item was damaged during shipping"
    },
    {
      "code": "OVERSTOCK",
      "description": "Customer returning excess inventory"
    }
  ]
}
```

| Field | Type | Description |
|-------|------|-------------|
| `reasonCodes` | array | List of reason code objects |
| `reasonCodes[].code` | string | Short code representing the return reason |
| `reasonCodes[].description` | string | Human-readable description |

## `GET /v1/codes/adjustment-reasons`

Retrieve all available inventory adjustment reason codes. Used when receiving returned items to classify the inventory adjustment.

### Response `200 OK`

```json
{
  "reasonCodes": [
    {
      "code": "CYCLE_COUNT",
      "description": "Adjustment from cycle count reconciliation"
    },
    {
      "code": "RECEIVING_ERROR",
      "description": "Correction for a receiving discrepancy"
    },
    {
      "code": "WAREHOUSE_DAMAGE",
      "description": "Item damaged in warehouse"
    }
  ]
}
```

| Field | Type | Description |
|-------|------|-------------|
| `reasonCodes` | array | List of reason code objects |
| `reasonCodes[].code` | string | Short code for the adjustment reason |
| `reasonCodes[].description` | string | Human-readable description |

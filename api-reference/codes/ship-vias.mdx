## `GET /v1/codes/ship-vias`

Retrieve all available shipping method codes. Used when creating RMAs and sales orders to specify the carrier and method.

### Response `200 OK`

```json
{
  "shipVias": [
    {
      "code": "UPS_GND",
      "description": "UPS Ground"
    },
    {
      "code": "FEDEX_2DAY",
      "description": "FedEx 2-Day Express"
    },
    {
      "code": "WILL_CALL",
      "description": "Customer Pickup / Will Call"
    }
  ]
}
```

| Field | Type | Description |
|-------|------|-------------|
| `shipVias` | array | List of ship via objects |
| `shipVias[].code` | string | Short code for the shipping method |
| `shipVias[].description` | string | Human-readable description |

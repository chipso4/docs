## `GET /v1/customers/contacts`

Retrieve all contacts for a specific customer.

<Note>
**V2 equivalent:** `GET /api/btu/v2/organization/{organizationId}/contacts` — returns [`CoreContact`](/data-types/contacts) entities.
</Note>

### Query parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `customerId` | string | **Yes** | The customer's unique identifier |

### Response `200 OK`

```json
{
  "contacts": [
    {
      "contactId": "CTX-9012",
      "firstName": "Jane",
      "lastName": "Smith",
      "emailAddress": "jane.smith@example.com"
    },
    {
      "contactId": "CTX-9013",
      "firstName": "Bob",
      "lastName": "Johnson",
      "emailAddress": "bob.johnson@example.com"
    }
  ]
}
```

| Field | Type | Description |
|-------|------|-------------|
| `contacts` | array | List of contact objects |
| `contacts[].contactId` | string | Unique identifier for the contact |
| `contacts[].firstName` | string | First name |
| `contacts[].lastName` | string | Last name |
| `contacts[].emailAddress` | string | Email address |

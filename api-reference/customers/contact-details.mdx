## `GET /v1/customers/contacts/details`

<Note>
**V2 equivalent:** `GET /api/btu/v2/contact/{contactId}` — returns a [`CoreContact`](/data-types/contacts) entity with full address and metadata.
</Note>

### Query parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `customerId` | string | **Yes** | The customer's unique identifier |
| `contactId` | string | **Yes** | The contact's unique identifier |

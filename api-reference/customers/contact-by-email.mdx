## `GET /v1/customers/contacts/by-email`

Find a customer contact by email address. Useful when only the email is known.

### Query parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `customerId` | string | **Yes** | The customer's unique identifier |
| `email` | string | **Yes** | The email address to search for |

### Response `200 OK`

Returns a single contact object matching the [`CoreContact`](/data-types/contacts) schema.

```json
{
  "contactId": "CTX-9012",
  "firstName": "Jane",
  "lastName": "Smith",
  "emailAddress": "jane.smith@example.com"
}
```

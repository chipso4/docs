## `GET /v1/customers`

Retrieve all customers. Returns an array of [`CoreOrganization`](/data-types/core-objects#coreorganization) entities with `organizationType: "Customer"`.

### Response `200 OK`

```json
{
  "customers": [
    {
      "organizationId": "C-827451",
      "organizationName": "Northern Manufacturing Inc.",
      "organizationType": "Customer",
      "isActive": true,
      "isCashAccount": false,
      "defaultBranchId": "WH-10",
      "paymentTermsCode": "NET30",
      "extendedAttributes": []
    },
    {
      "organizationId": "C-330092",
      "organizationName": "Pacific Supply Co.",
      "organizationType": "Customer",
      "isActive": true,
      "isCashAccount": false,
      "defaultBranchId": "WH-20",
      "paymentTermsCode": "NET30",
      "extendedAttributes": []
    }
  ]
}
```

Each item in the `customers` array follows the [`CoreOrganization`](/data-types/core-objects#coreorganization) schema. The list endpoint returns a summary view — use [`GET /v1/customers/details`](/api-reference/customers/customer-details) for the full entity.

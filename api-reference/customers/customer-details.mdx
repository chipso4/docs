## `GET /v1/customers/details`

Retrieve full details for a specific customer. Returns a [`CoreOrganization`](/data-types/core-objects#coreorganization) entity with primary address, primary contact, and all fields.

### Query parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `customerId` | string | **Yes** | The customer's unique identifier |

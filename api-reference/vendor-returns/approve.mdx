## `PATCH /v1/vendor-return/approve`

Approve a vendor return for warehouse processing. Moves status from DRAFT to APPROVED.

### Request body

```json
{
  "returnNumber": "VR-2026-00050"
}
```

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `returnNumber` | string | **Yes** | The vendor return number |

### Response `200 OK`

```json
{
  "returnNumber": "VR-2026-00050",
  "messages": ["Vendor return approved for processing."]
}
```

## `PATCH /v1/vendor-return/ship`

Record that a vendor return has been shipped back to the manufacturer. Includes tracking information.

### Request body

```json
{
  "returnNumber": "VR-2026-00050",
  "trackingNumber": "1Z999AA10123456784",
  "carrierCode": "UPS",
  "shipDate": "2026-03-18T00:00:00Z"
}
```

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `returnNumber` | string | **Yes** | The vendor return number |
| `trackingNumber` | string | No | Carrier tracking number |
| `carrierCode` | string | No | Carrier code (e.g., `"UPS"`, `"FEDEX"`) |
| `shipDate` | datetime | No | Date of shipment |

### Response `200 OK`

```json
{
  "returnNumber": "VR-2026-00050",
  "messages": ["Vendor return shipped. Tracking: 1Z999AA10123456784"]
}
```

## `PATCH /v1/vendor-return/picked`

Record that warehouse staff have physically picked the defective items for the vendor return.

### Request body

```json
{
  "returnNumber": "VR-2026-00050"
}
```

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `returnNumber` | string | **Yes** | The vendor return number |

### Response `200 OK`

```json
{
  "returnNumber": "VR-2026-00050",
  "messages": ["Vendor return marked as picked."]
}
```

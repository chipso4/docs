## `GET /v1/return`

Retrieve full details for a specific RMA. Returns a [`CoreRma`](/data-types/core-objects#corerma) entity with line items, account locations, and credit details.

### Query parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `RmaNumber` | string | **Yes** | The RMA number |

The response follows the full [`CoreRma`](/data-types/core-objects#corerma) schema. Key nested types:

| Field | Type | Description |
|-------|------|-------------|
| `billTo` | [`CoreAccountLocation`](/data-types/core-objects#coreaccountlocation) | Bill-to account and address |
| `shipTo` | [`CoreAccountLocation`](/data-types/core-objects#coreaccountlocation) | Ship-to account and address |
| `returnTo` | [`CoreAccountLocation`](/data-types/core-objects#coreaccountlocation) | Where items should be returned |
| `endCustomer` | [`CoreAccountLocation`](/data-types/core-objects#coreaccountlocation) | End customer (warranty flow) |
| `items` | [`CoreRmaReturnItem[]`](/data-types/core-objects#corermareturniitem) | RMA line items with invoice linkage and credit details |

See the [core objects reference](/data-types/core-objects) for full field listings.
